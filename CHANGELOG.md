## [1.1.0] - 2026-02-22

### Added
- **`@mcp-server-starter.init`** command - Initialize new MCP server projects with complete setup
  - 17 detailed implementation steps
  - Complete file templates for all configuration files
  - Dual export pattern (standalone + factory)
  - Example hello_computer tool
  - Git initialization and dependency installation
  - Comprehensive verification and troubleshooting
- **`@mcp-server-starter.add-tool`** command - Generate new tool scaffolding
  - 8 detailed implementation steps
  - Complete tool template with TypeScript interfaces
  - Automatic server registration (standalone + factory)
  - Test file generation with 4 test cases
  - CLI and natural language argument support
  - 5 usage examples and troubleshooting guide

### Changed
- Updated package.yaml with 2 new commands
- Updated progress tracking: Milestone 2 now 50% complete (2/4 tasks)
